Detailed Description

Abstract class defining how an input stream behaves.

Input streams give you the ability to read data sequentially. The base class supplies an implementation of the popular stream operators << and >>. These should be used with care as error handling is not easily implemented.

virtual int16_t stm32plus::InputStream::read ( ) [pure virtual]

Read a byte from the stream, potentially blocking until data is available.

Returns:
If the read succeeds then the byte is returned in the lower 8 bytes, hence the return value is in the range 0..255. If the read fails then the return value is StreamError. If the end of the stream has been reached then the return value is EndOfStream.

Implemented in stm32plus::CircularBufferInputOutputStream, stm32plus::FileInputStream, stm32plus::LinearBufferInputOutputStream, stm32plus::LzgDecompressionStream, stm32plus::UsartInputOutputStream, and stm32plus::UsartInputStream.
