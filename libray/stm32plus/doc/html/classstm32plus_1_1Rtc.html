Detailed Description

Real time clock abstraction.

The clock source is assumed to be the LSE 32.768kHz clock. The ticker is not output on the tamper pin. The user must never declare more than one instance of this class to be in scope at any one time due to the use of a callback instance pointer for the interrupt handler.
