Constructor & Destructor Documentation

stm32plus::SdioSecureDigitalCard::SdioSecureDigitalCard ( )

Constructor. Enables the SDIO IRQ, initialises GPIO and clocks and initialises the peripheral. Note that this constructor can set an error in the errorProvider if initialisation fails. Callers should check the error status before attempting to use this.
