Detailed Description

Implementation of an input stream for the serial USART.

The stream methods can be used to read data from the USART in polling mode

int16_t stm32plus::UsartInputStream::read ( ) [virtual]

Read a byte from the stream, potentially blocking until data is available.

Returns:
If the read succeeds then the byte is returned in the lower 8 bytes, hence the return value is in the range 0..255. If the read fails then the return value is StreamError. If the end of the stream has been reached then the return value is EndOfStream.
