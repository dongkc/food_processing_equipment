Detailed Description

Implementation of a USART observer that listens for data and writes it to an output stream.

This utility class provides a bridge between incoming data from a USART and an output stream that can receive it. Data is read asynchronously and passed to the output stream.

The writer is called under the interrupt handler so it is critical that the writer does not call anything that depends on interrupts being enabled, such as MilisecondTimer::delay (which depends on SysTick firing)
