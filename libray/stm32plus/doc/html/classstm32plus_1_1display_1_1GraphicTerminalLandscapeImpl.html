Detailed Description

template<class TGraphicsLibrary>
class stm32plus::display::GraphicTerminalLandscapeImpl< TGraphicsLibrary >

Implementation of a terminal for LCDs in landscape mode. In this mode hardware scrolling is not supported and the terminal will clear down and go back to the top when the bottom is reached.
